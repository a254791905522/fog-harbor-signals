# Shrouded Harbor Signals Release Package

App name: Shrouded Harbor Signals

Bundle ID: com.rosewood.fogharbor.game

Device support: iPhone only

Gameplay fact card:
- Core action: Guide ships through fog by replying with the correct signal (Foghorn, Lamp, Flag, Ack, Warn, SOS).
- Main puzzle: Read each ship's intent and match the correct signal before patience runs out.
- Weather system: Fog, HeavyRain, Thunder, and Blizzard reduce visibility across 50 levels.
- Progression: 50 levels across 5 historic lighthouse stations (Portland Head, Boston Harbor, Cape Cod, Sandy Hook, Cape Ray).
- Monetization and network: offline, no ads, no in-app purchases, no account, no analytics, no tracking.
